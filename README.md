# CPP_Tutorial

A comprehensive C++ learning repository where I document everything I learn - from the fundamentals to advanced topics. This repository  contains notes, examples, and mini-projects created throughout  my journey of knowing C++ better.

## Prerequisites

- C++ compiler (GCC, clang, or MSVC)
- Visual Studio Code (or any C++ IDE)
- Git (Optional)

## Compile and Run

### Linux / macOS

```bash
g++ filename.cpp -o program
./program
```

### Windows

```cmd
g++ filename.cpp -o program
program.exe
```
